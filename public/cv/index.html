CV - Jonathan van der Ceelen

Frontend developer

Jeugd

Mijn verhaal begint in Tilburg, waar ik opgroeide en tot mijn 22e woonde. Ik heb in Tilburg de Vrije Basisschool doorlopen en ben daarna overgestapt op een reguliere middelbare school, waarbij ik het profiel Natuur en Gezondheid combineerde met het gymnasium. Met name de oude verhalen en mythes vond ik erg interessant.

Studie

Op mijn 22e ben ik gaan studeren in Amsterdam: Oudheidkunde. Ik ging achter mijn interesse voor oude verhalen en mythes aan. Ik heb acht jaar gestudeerd, vijf jaar bachelor, drie jaar master. Uiteindelijk studeerde ik in de zomer van 2013 af op het Gilgamesh Epos - een van de oudste geschreven verhalen ter wereld - en kon het echte leven beginnen.

Werk

Na acht jaar vooral theoretisch bezig te zijn geweest binnen mijn studie, vond ik dat het tijd was voor meer praktijk. Daarnaast wilde ik mijn creativiteit kwijt kunnen. Ik koos voor web development en na een jaar van online cursussen volgen en solliciteren werd ik door een kleine startup opgepikt als junior developer. Hier startte mijn carriere.

2015 - huidig

Frontend web developer - DGN Publishers

DGN Publishers beheert een drietal websites waarbij Hypotheken, notarisaktes en in het seizoen zorgverzekeringen worden aangeboden en vergeleken. Bij DGN kwam ik binnen als frontend dev. Ik had een gemiddelde kennis van html, ik kon prima omgaan met Bootstrap, ik wist Git te gebruiken vanaf de command line en eenvoudige queries schrijven in MySql. In de ruim drie jaar dat ik bij DGN werkzaam ben heb ik deze vaardigheden uitgebreid en een hoop nieuwe dingen geleerd. Ik probeer waar mogelijk en zinvol gebruik te maken van HTML5 tags, ben afgestapt van Bootstrap en heb geleerd alle css zelf te schrijven, heb mezelf jQuery aangeleerd en probeer waar mogelijk zoveel mogelijk vanilla JS te gebruiken en heb mezelf wegwijs gemaakt binnen PHP.

2014 - 2015

Web developer - TruQu

TruQu is een bedrijf dat software maakt om professionals te reviewen. Dit was mijn eerste baan als Web Developer. Op dat moment had ik slechts een basale kennis van een relatief groot aantal talen, scripts en frameworks. Html, Css, Javascript, Python, Erlang, Elm, Java, Django. Ik leerde met Linux omgaan, Git gebruiken, queries schrijven in MySql, emails maken in Mailchimp, Bootstrap gebruiken en nog een hele hoop andere dingen. Ik kwam er al gauw achter dat ondanks dat ik het meeste affiniteit had met de frontend. Data ophalen was leuk, maar de manier waarop deze data gepresenteerd werd was interessanter. Hierbij kon ik mijn creativiteit kwijt en mijn eigen inbreng over wat ik mooi en minder mooi vond.
